<h1 align="center">Lunaria</h1>

<p align="center">
  <strong>Desktop-first AI development environment</strong>
</p>

---

Lunaria is a desktop-first AI development environment built for developers who want full control over their tooling. It runs locally as a native desktop application, with a plugin and extension system that exposes every part of the platform for customization. Mobile support is provided via a companion app.

## Repositories

- **[lunaria](https://github.com/LunariaAi/lunaria)** — Main monorepo: Tauri desktop app, React UI, Rust backend
- **[extension_boilerplate](https://github.com/LunariaAi/extension_boilerplate)** — Build UI extensions with any framework
- **[plugin_boilerplate](https://github.com/LunariaAi/plugin_boilerplate)** — Build backend plugins with hooks and lifecycle events

## Tech Stack

- **Desktop**: Tauri v2, React 19, TypeScript, Rust
- **Mobile**: Expo (React Native)
- **Package Manager**: Bun
